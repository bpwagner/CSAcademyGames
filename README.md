# CSAcademyGames
Some simple games I wrote for CS Academy Sandbox

You can use these for any class you teach.  
I really like the CS Academy environment for teaching Python coding
https://academy.cs.cmu.edu/

Bounce - get a ball bouncing
Bounce2 - add a paddle to the Bouncee program
Bounce3 - Add score keeping to Bounce game

Pong - two person pong game, similar to bounce

Dino - Like the jump dino game when you have no internet on Chrome

ShapeJumper - Jump the rolling cube over spikes

Flappy - Easy Flappy bird clone
